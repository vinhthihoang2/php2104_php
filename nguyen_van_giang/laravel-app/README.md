## How to run project

Run command bellow:

```
cd nguyen_van_giang/laravel-app/
composer install
npm install
npm run dev
php artisan migrate --seed
```

And start server

```
php artisan serve
```

Run command bellow if error
```
composer require "almasaeed2010/adminlte=~3.1"
npm run dev
```

Now, you can try it http://localhost:8000/home-page
